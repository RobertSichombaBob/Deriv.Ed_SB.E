# CBU Math & Econ - 6-Week Intensive Program

A complete GitHub Pages website for the Mathematical Methods 1 and Economics 110 intensive program at The Copperbelt University.

## Features

- **Responsive Design**: Works on all devices (desktop, tablet, mobile)
- **Course Management**: Separate sections for Mathematics and Economics
- **User Management**: Registration, login, and student dashboard
- **Resource Distribution**: Organized notes, problems, and solutions
- **Interactive Schedule**: Weekly breakdown with resource links
- **FAQ Section**: Comprehensive question and answer database
- **Team Information**: Meet the teaching staff
